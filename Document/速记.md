# 第一章 SOC 基础
## 1. SOC Workflow 顺序
### 题型
SOC 工作流的正确顺序是什么？

### 正确答案
Collect → Ingest → Validate → Document → Report → Respond

### 解析
+ Collect：收集日志与事件
+ Ingest：导入 SIEM
+ Validate：验证真实性
+ Document：形成记录
+ Report：提交报告
+ Respond：执行响应

### 高频记忆
“收集 → 导入 → 验证 → 文档 → 报告 → 响应”

### 常考点
| 角色 | 职责 |

# 第二章 Windows 日志与 Event ID
## 必背 Event IDs
| Event ID | 含义 |
| --- | --- |
| 4624 | Successful Logon |
| 4625 | Failed Logon |
| 4688 | Process Creation |
| 4740 | Account Locked |
| 5140 | File Share Access |
| 4657 | Registry Modified |
| 7045 | Service Installed |


---

## 典型题：4688
### 题型
Splunk 中监控进程创建使用哪个 Event ID？

### 正确答案
4688

### 解析
4688 表示：A new process has been created。 SOC 中常用于：

+ PowerShell 执行
+ cmd 调用
+ 可疑子进程
+ 横向移动

---

## 典型题：4740
### 题型
Windows Event ID 4740 表示什么？

### 正确答案
A user account was locked out

### 解析
通常表示：

+ 密码爆破
+ AD 锁定策略触发
+ 异常登录

Operational Threat Intelligence

### 解析

+ 攻击活动

# 第四章 攻击识别（最重要）
## SQL Injection
### 典型特征
+ ' OR 1=1 --
+ %27
+ union select

### 正确答案
SQL Injection

### 防御
+ 参数化查询
+ Prepared Statement
+ Safe API

---

## XSS
### 特征
+ %3Cscript%3E

### 正确答案
Cross-Site Scripting (XSS)

### 防御
+ HTML Encode
+ 输入过滤
+ CSP

---

## Directory Traversal
### 特征
+ ../
+ %2E%2E/
+ /etc/passwd

### 正确答案
Directory Traversal

### 解析
攻击者尝试跳出 Web Root 访问系统文件。

---

## Parameter Tampering
### 场景
把：

/debit=100

改成：

/debit=10

### 正确答案
Parameter Tampering

### 解析
修改 URL 参数或隐藏字段。

---

## DNS Exfiltration
### 特征
+ 大量 TXT
+ NULL payload
+ 异常 DNS 查询

### 正确答案
DNS Exfiltration

### 解析
通过 DNS 偷数据。

---

## Ransomware
### 特征
+ 文件后缀突然变化
+ 文件重命名暴增
+ 大量加密行为

### 正确答案
Ransomware Attack

# 第十三章 高频纠错题
