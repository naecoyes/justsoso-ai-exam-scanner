📖 高频不常见词汇 — 读题关键词锁定考点

Gruesome adj.可怕的 → 描述破坏性极强的恶意软件
Inundate v.淹没/应接不暇 → 海量伪造请求攻击服务器
Tampering n.篡改 → Parameter Tampering攻击URL参数
Obfuscated adj.混淆的 → 恶意代码加密混淆逃避AV检测
Tautology n.重言式/恒真式 → SQL注入中 1=1 或 'OR T=T
Starvation n.饥饿/耗尽资源 → DHCP Starvation耗尽IP池
Adversary n.对手/敌手 → CTI中替代"hacker"的正式用词
Exfiltration n.数据渗出 → 攻击者将数据偷传出内网
Reconnaissance n.侦察/踩点 → Kill Chain第1步
Eradication n.根除 → IH&R第4阶段，清除Root Cause
Triage n.分流/分类 → 告警验证与严重性定级（L1职责）
Custody n.保管/羁押 → Chain of Custody 证据监管链
Deprovisioning n.撤销权限 → SOAR Playbook自动停用用户
Decoy n.诱饵 → Honeypot蜜罐的作用描述
Heterogeneous adj.异构的 → SIEM接收不同格式/来源的日志
Dissemination n.分发/传播 → 威胁情报生命周期/安全标准传播
In-house adj.内部的 → 企业自建SOC，非外包
Absorbing v.吸收 → DDoS时增加带宽+容量=Absorbing the Attack
解题技巧：看到 Custody=取证监管链 | Triage=事件分流分类 | Exfiltration=数据外泄 | Adversary=攻击者 | Tautology=SQL注入恒真式 | Obfuscated=恶意代码混淆
